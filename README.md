## 📖 About the Project

**Edutrip Academy** is an online teacher training website developed as a university group project. The platform provides educators with access to professional development courses, learning materials, instructor guidance, and certification opportunities.

The system focuses on creating an engaging learning experience while demonstrating full-stack web development concepts using HTML, CSS, JavaScript, PHP, and MySQL.
